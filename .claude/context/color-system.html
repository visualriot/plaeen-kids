Design System v1.0 — Color Tokens

Color System
Plaeen

WCAG 2.1 compliant color tokens for a forced dark-mode platform. All foreground colors verified against bg-base (#0A0514).

Base bg #0A0514 Accent green #76E900 Accent purple #2D1B4D Mode Dark only Standard WCAG 2.1
AAA ≥ 7:1 — any text size
AA ≥ 4.5:1 — normal text
AA Large ≥ 3:1 — 18px+ bold or 24px+
Non-text only — decorative / disabled

#0A0514 · purple-950
Root page background. Never used as foreground.
Cards / panels
--surface-raised
#130D21
Cards, sidebars, section containers.
Dropdowns / tooltips
--surface-overlay
#1D1535
Dropdowns, tooltips, inline popups.

--surface-floating

03 — Text Tokens
Primary text — Aa
--text-primary
#F2EEF9 · neutral-50
AAA 17.4:1 vs surface-base
Headings, body copy, UI labels.
Secondary text — Aa
--text-secondary
#BAAFD9 · neutral-200
AAA 9.1:1 vs surface-base
Subtitles, helper text, input placeholders.
Muted text — Aa
--text-muted
#8B7AAA
AA 5.2:1 vs surface-base
Timestamps, captions, metadata. Use ≥ 14px.
Disabled — Aa
--text-disabled
#4A3D6A
non-text 2.3:1 — intentional
Disabled inputs/buttons only. Never informational.
Inverse — Aa
--text-inverse
#0A0514 · purple-950
AAA 12.9:1 vs green-400
Text on primary green backgrounds/buttons.
Accent green — Aa
--text-accent
#76E900 · green-400

Links, highlights, active states, CTAs.
Accent purple — Aa
--text-accent-secondary
#A07FD0
